# light-ui

> a Vue.js Mobile UI Toolkit

## 组件分类

![component](README/component.png)

+ basic 基础组件

+ form 表单组件

+ Popup 弹窗组件

+ Scroll 滚动组件

## Build Setup

``` bash
# install dependencies
npm install

# 运行 npm 包开发使用
npm run dev

# 打包 npm 包
npm run build

# 运行开发UI组件，以及说明文档文件 --- 官网 cube.com
npm run doc-dev

# 只打包说明文档文件
npm run doc-build

# 只打包UI组件的demo
npm run demo-build

# 打包说明文档文件 和 打包UI组件的demo
npm run doc-demo-build
```
