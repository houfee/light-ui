## Build Setup

``` bash
# install dependencies
npm install

# 运行 npm 包开发使用
npm run dev

# 打包 npm 包
npm run build

# 运行开发UI组件，以及说明文档文件 --- 官网 cube.com
npm run doc-dev

# 只打包说明文档文件
npm run doc-build

# 只打包UI组件的demo
npm run demo-build

# 打包说明文档文件 和 打包UI组件的demo
npm run doc-demo-build
```

# 这个库是 EventSource客户端的纯JavaScript实现。 API的目标是兼容 W3C。
